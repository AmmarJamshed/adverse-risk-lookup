# ARL Obligations Workflow Tutorial Script

Video: **`ARL_Beginner_How_To_Use.mp4`**  
Source: browser recording of https://arl-adverse-risk-lookup.netlify.app  
Rebuild: `python scripts/record_live_tutorial.py`

Login: `admin@arl.local` / `ChangeMe123!`

## Chapters

1. **Sign in** — demo credentials → Continue to console  
2. **Horizon Scan** — KPIs, jurisdiction filter, open an instrument  
3. **Assess candidate** — Mark applicable → create obligation & case  
4. **Applicability** — inbox for under-review items  
5. **Obligations Register** — structured requirements table  
6. **Gap Analysis Cases** — list and open a case  
7. **Case workspace** — map policy/control coverage  
8. **Policies & Controls** — library inventory  
9. **Regulatory Sources** — FCA, FinCEN, EBA, SBP, Fed  
10. **Administration** — users and roles  

Practice: Scan → Assess → Register → Map gaps.
